# 🎬 Movie Ticket Booking System

## Overview

A console-based Movie Ticket Booking System developed using C# and .NET. The application allows customers to view movies, book tickets, make payments, receive notifications, and view booking history. The project demonstrates Object-Oriented Programming concepts, file handling, exception handling, interfaces, and dependency injection.

## Features

* View Movies
* View Customers
* Book Movie Tickets
* Wallet Balance Validation
* UPI Payment
* Card Payment
* Email Notifications
* Booking History Management
* File Handling using Text Files
* Exception Handling
* Menu Driven Console Application

## Technologies Used

* C#
* .NET Console Application
* Object-Oriented Programming (OOP)
* File Handling

* Dependency Injection
* Interfaces and Polymorphism
